SciDB Non-root Installer

This is Paradigm4's non-root installer. It is used to install SciDB Community or Enterprise Edition when you do not have root access to the SciDB hosts. It can provide PostgreSQL if needed and it provides the SciDB shim.

Presently the non-root installer (named nrinstall) only supports CentOS 6 and RedHat 6 and SciDB version 13.12 onwards.

The computer where you run nrinstall (called the "installer computer") does not have to be one of the hosts you will be running SciDB on. You will run nrinstall as a regular user on the installer computer.

To get started download the following tarfile: SciDB_nrinstall.tar onto the installer computer and untar it. A directory "SciDB_nrinstall" will be the result. cd into that directory and type: "./nrinstall"

The order of events should be something like this:

  1. tar -xf SciDB_nrinstall.tar
  2. cd SciDB_nrinstall
  3. ./nrinstall -i
  4. If you do not have write access to an existing PostgreSQL installation on the coordinator
    • run the setupPostgreSQL script that is provided by the non-root installer nrinstall on the SciDB coordinator host.
  5. logout and log back into the coordinator
  6. scidb.py init_syscat [db] (notice scidb.py is now in your PATH)
  7. scidb.py initall [db]
  8. scidb.py startall [db]
  9. Do your work
  10. scidb.py stopall [db]
